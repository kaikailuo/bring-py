## 一、顺序结构复习

### 1. 概念说明
顺序结构是程序中最基本的一种结构。  
程序按照**自上而下的顺序**依次执行语句。  
每条语句只会被执行一次，不存在条件判断或循环。  

任何计算机程序都是为了执行特定任务。  
程序一般包括三个步骤：

- **输入**：接收用户或文件输入的数据；  
- **处理**：程序对输入的数据进行计算或逻辑处理；  
- **输出**：将结果显示或保存给用户。  

### 2. 流程图示例

```
输入 → 处理 → 输出
```

### 3. 案例一：求一元一次方程的根

方程形式：  
\( ax + b = 0 \)

输入：a, b  
处理：计算 x 的值  
输出：打印结果  

**示例代码：**
```python
a = float(input("请输入 a："))
b = float(input("请输入 b："))
x = -b / a
print("方程的根为：", x)
```

## 三、分支（选择）结构

### 1. 概念说明

**分支结构**根据条件的真假来选择不同的执行路径。  
这是程序实现自动决策能力的关键。  

### 2. if 语句格式

#### （1）单分支 if 语句
```python
if 条件:
    语句块1
```
> 若条件成立，则执行语句块1，否则跳过。

#### （2）双分支 if-else 语句
```python
if 条件:
    语句块1
else:
    语句块2
```
> 若条件成立，执行语句块1；否则执行语句块2。

### 3. Python 比较（关系）运算符

| 运算符 | 含义 | 示例 |
|--------|------|------|
| `==` | 等于 | `a == b` |
| `!=` | 不等于 | `a != b` |
| `>` | 大于 | `a > b` |
| `<` | 小于 | `a < b` |
| `>=` | 大于等于 | `a >= b` |
| `<=` | 小于等于 | `a <= b` |

示例：  
```python
a, b = 10, 20
if a < b:
    print("a 小于 b")
```

还可以使用连续比较：  
```python
x = 15
if 10 <= x <= 20:
    print("x 在 10 和 20 之间")
```

## 五、案例：一元二次方程求根

方程形式：  
\( ax^2 + bx + c = 0 \)

### 1. 分析思路
1. 输入 a, b, c；  
2. 计算判别式 \( \Delta = b^2 - 4ac \)；  
3. 根据 Δ 的值判断：  
   - 若 Δ > 0：有两个不同实根  
   - 若 Δ = 0：有一个实根  
   - 若 Δ < 0：无实根  

### 2. 示例代码
```python
import math

a = float(input("请输入 a："))
b = float(input("请输入 b："))
c = float(input("请输入 c："))

d = b**2 - 4*a*c

if d > 0:
    x1 = (-b + math.sqrt(d)) / (2*a)
    x2 = (-b - math.sqrt(d)) / (2*a)
    print("两个实根：", x1, "和", x2)
elif d == 0:
    x = -b / (2*a)
    print("一个实根：", x)
else:
    print("此方程无实数解")
```

### 3. 测试数据
| 输入 (a,b,c) | 输出 |
|---------------|-------|
| (2,5,3) | 两个根：-1.0 和 -1.5 |
| (2,4,2) | 一个根：-1.0 |
| (2,4,3) | 无解 |

## 六、总结

- **顺序结构**：按顺序执行；  
- **分支结构**：根据条件执行不同语句；  
- **常见语句组合：**  
  - 输入 → 判断 → 输出  
  - if / else 语句是程序决策的关键；  
  - 注意类型转换（`int()`、`float()`）。
