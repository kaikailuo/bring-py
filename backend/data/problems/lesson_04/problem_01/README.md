# 例题4-1：计算心脏恢复率

### 题目描述

运动后心脏恢复率是一个有效的健康标准，某心率监测设备通过记录使用者运动刚结束后 1 分钟内的心率数 a，和停止运动 1 分钟后的心率数 b，计算心脏恢复率 ( r = (a-b)/10 )。
如果 r ≥ 3，则表示心脏恢复率良好，否则表示心脏恢复率不佳。请编写程序，根据输入 a 和 b，输出心脏恢复率和心脏恢复状况。

### 输入格式

输入包含两行：
第一行输入整数 a（运动刚结束后的心率）
第二行输入整数 b（停止运动 1 分钟后的心率）

### 输出格式

输出一行：心脏恢复率 r 和状况（良好/不佳），用空格分隔，r 保留一位小数

### 输入样例

```
161
113
```

### 输出样例

```
4.8 良好
```
