# 例题3-1：统计合格比例与最大值

### 题目描述

在技术与设计课程中，同学们要完成一个桥梁承重的项目活动。  
承重量大于 250 克，就算承重量指标合格。  
请编写程序，根据用户输入的 n 个小组的桥梁承重量，计算合格作品的百分比（保留一位小数）并找出最大承重量。  
要求：使用循环结构。

### 输入格式
输入包含 n+1 行：  
第一行输入一个整数 n，表示桥梁作品数量  
接下来的 n 行，每行输入一个整数，表示每个桥梁的承重量

### 输出格式
输出两行：  
第一行输出合格作品百分比  
第二行输出最大承重量

### 输入样例
```

4
356
253
180
395

```

### 输出样例
```

75.0
395

````
