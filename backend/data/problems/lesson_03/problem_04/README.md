# 例题3-4：统计到校人数与所需教室

### 题目描述

欢迎小学生到中学进行入学准备参观，因每天接待多所学校，需要统计当天到校人数，并按照每 40 人安排教室作为休息室。  
请编写程序，输入学校数量及每所学校到访人数，输出当天总人数及所需教室数。  
要求：使用循环结构。

### 输入格式
输入包含 n+1 行：  
第一行输入一个整数 n，表示学校数量  
接下来的 n 行，每行输入一个整数，表示每所学校到访人数

### 输出格式
输出两行：  
第一行输出当天总人数  
第二行输出所需教室数

### 输入样例
```

3
175
231
120

```

### 输出样例
```

526
14

````
