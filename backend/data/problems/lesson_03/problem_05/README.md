# 例题3-5：统计可保留文创产品

### 题目描述

学校的学生创业公司将今年文创比赛获奖产品生产进行销售，销量比去年平均销售量 125 件高的产品可继续保留生产。
请编写程序，输入文创产品种类数量及每种产品的销量，输出今年销量最大的文创产品销量和可保留生产的产品种类数。
要求：使用循环结构。

### 输入格式

输入包含 n+1 行：
第一行输入一个整数 n，表示文创产品种类数量
接下来的 n 行，每行输入一个整数，表示每种产品销量

### 输出格式

输出两行：
第一行输出今年销量最大的文创产品销量
第二行输出可以保留生产的文创产品种类数

### 输入样例

```
5
133
77
98
210
183
```

### 输出样例

```
210
3
```
