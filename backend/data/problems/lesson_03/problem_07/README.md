# 例题3-7：计算愿望数

### 题目描述

小明的妈妈让小明在学习之余去社区图书馆帮忙，为了鼓励他坚持这项活动，他每去一次妈妈都会根据他活动的时间和完成任务发放一定数量的愿望星，当攒够了一定的愿望星，妈妈就能满足他一个合理的愿望。请帮助小明写一个程序，通过他去年12个月每个月得到的愿望星数目，计算他去年一共获得多少颗愿望星，平均每月获得了多少？（要求用循环结构解决该问题，平均数结果保留一位小数）。

## 输入样例
```
1月获得的愿望星个数是：7
2月获得的愿望星个数是：5
3月获得的愿望星个数是：11
4月获得的愿望星个数是：3
5月获得的愿望星个数是：6
6月获得的愿望星个数是：4
7月获得的愿望星个数是：9
8月获得的愿望星个数是：6
9月获得的愿望星个数是：10
10月获得的愿望星个数是：8
11月获得的愿望星个数是：12
12月获得的愿望星个数是：9
```

## 输出样例
```
去年小明一共获得愿望星 90 颗
平均每个月获得的愿望星是： 7.5 颗
```
